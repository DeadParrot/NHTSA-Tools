PROGRAM FUNCTION AND METHODS:

	MaxDiff computes the maximum difference between pairs of UDS y-series
	files.

PROGRAM FEATURES/OPTIONS:

	Any y-series UDS files can be used.  The two files are checked for
	consistent DEL step values and units.

KNOWN PROBLEMS/CONSTRAINTS:

	None.
