PROGRAM FUNCTION AND METHODS:

	UDSDump provides a display of UDS file specifications and/or
	measurement values to the screen or a user-specified file.

PROGRAM FEATURES/OPTIONS:

	A single UDS specification field, the full specifications, or the
	full specifications and measurment values can be displayed by UDSDump.

	UDSDump output can be displayed to the screen or output to a specified
	file.

	UDSDump output is tailored to the type of file.  Groups of related
	fields with blank/zero key values are not displayed.

SETUP/RUN INSTRUCTIONS:

	An existing UDS file must be supplied to the program.

KNOWN PROBLEMS/CONSTRAINTS:

	None.
